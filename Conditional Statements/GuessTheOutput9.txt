What is the output of the following code snippet?


#include <iostream>
using namespace std;
int main() {
    int x = 10;
    int y = 5;
    cout << (x > 5 && y < 3);
}


Select one of the following options:

- 1
- 0
- Throws an error
- None of the above



Correct Answer: 0

Explanation: && returns false if either operand is false
