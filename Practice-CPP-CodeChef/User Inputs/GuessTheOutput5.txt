What is the output of the following code?


#include <iostream>
using namespace std;
int main() {
    string name;
    cin >> name;
    cout << name.length();
}


Select one of the following options:
- Prints the length of the name
- Prints the first character of the name
- Prompts user to enter two names and prints the longer name
- Results in a compilation error


Correct Answer: Prints the length of the name

Explanation: The program prompts the user to enter a string, reads it using cin, and then prints the length of the entered string
