What is the output of the following code? cout << 7.0 / 2;


Select one of the following options:

- 3.5
- 3
- 2.5
- 4


Correct Answer: 3.5

Explanation: When dividing a decimal value by an integer the result is a decimal value. As 7.0 is a double value the result will be 3.5, if the value would have been 7(integer value) the result would be 3.
