What does the following code represent?


#include <iostream>
using namespace std;
int main() {
    int age = 25;
}


Select one of the following options:

- Printing the age variable
- Storing the value 25 in the age variable
- Declaring the age as a constant
- Initializing a boolean variable


Correct Answer: Storing the value 25 in the age variable

Explanation: The code int age = 25; initializes the variable age with the value 25.
