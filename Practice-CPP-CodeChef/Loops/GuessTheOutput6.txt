What does the following code snippet do?


for (int i = 0; i < 3; i++) {
    cout << "Hello ";
}



Select one of the following options:

- Prints "Hello" three times
- Prints "Hello" once
- Prints "Hello" four times
- Prints "Hello" twice


Correct Answer: Prints "Hello" three times

Explanation: The loop prints "Hello" three times as the value of i goes from 0 to 3 and when the value becomes equal to 3 the loop is exited
