How do you print the numbers 1 to 5 on the same line with spaces?


Select one of the following options:

A) cout << 1 << " " << 2 << " " << 3 << " " << 4 << " " << 5;
B) cout << "1 2 3 4 5";
C) Both A and B
D) None of the above


Correct Answer: C) Both A and B

Explanation: " " can be used to separate multiple outputs on the same line, or they can be printed as a single string
