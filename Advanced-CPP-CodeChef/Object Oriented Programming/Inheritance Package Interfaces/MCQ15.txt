Which of the following statements about abstract classes is true? Select the correct option.

1.  An abstract class cannot have constructors.

2.  An abstract class cannot be used as a base class for other classes.

3.  An abstract class can have both pure virtual and concrete member functions.

4.  An abstract class must have all its member functions declared as pure virtual.



Answer: 3.  An abstract class can have both pure virtual and concrete member functions.
