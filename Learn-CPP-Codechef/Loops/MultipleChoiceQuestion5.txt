How many times will "C++" be printed by this code?

#include <iostream>
using namespace std;

int main() {
    for (int i = 1 ; i <= 5 ; i = i + 1) {
        cout << "C++"<< endl;
    }
}


Select one of the following options.

1.  4
2.  5
3.  6
4.  Doesn't stop printing


Answer: 5
