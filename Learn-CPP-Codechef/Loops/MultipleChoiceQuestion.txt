What is the output of this code?

#include <iostream>
using namespace std;

int main() {
    int a = 4;
    int b = 1;
    while ( a != b ) {
        cout << b << " ";
        b = b+1;
    } 
}



Select one of the following options.

1.  1 2 3 4
2.  1 2 3 4 5
3.  1 2 3
4.  Keeps printing all integers from 1 to infinity.



Answer: 1 2 3
